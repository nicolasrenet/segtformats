# SegtFormats: A Toolbox for Manipulating Segmentation Metadata


## Installation


  ```bash
  pip install segtformats
  ```

## Features



## Command-line utilities

+ `alto_to_page_xml`: Alto → Page conversion.
+ `alto_to_json`: Alto → JSON conversion.
+ `json_to_page`: JSON → Page conversion.
+ `page_xml_to_json`: Page → JSON conversion.
+ `anyseg_to_json`: Detect input format and convert to JSON.
+ `json_to_json`: Various transformation on JSON metadata, including repairs (region boundaries, line-to-region assignments).
+ `anyseg_to_ascii`: Render segmentation metadata on the terminal.
